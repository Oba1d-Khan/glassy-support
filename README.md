## üëã Welcome

This repository is the central hub for public support for the **Glassy New Tab** extension. While the source code for Glassy remains private to protect the project's integrity, we use this space to track bugs, discuss feature requests, and share updates transparently.

### üêõ How to Report a Bug

Found something broken? Please help us fix it by [opening a new issue](https://github.com/Oba1d-Khan/glassy-support/issues/new).

**When reporting a bug, please include:**
1. **Description:** What happened?
2. **Steps to Reproduce:** How can we make it happen again?
3. **Screenshots/Video:** Visuals help immensely!
4. **Browser/OS:** (e.g., Chrome v120 on Windows 11)

## ‚ú® About Glassy

Glassy transforms your browser's blank new tab into a calm, powerful, and personal command center. It is designed to be fast, beautiful, and 100% private.

- **Stunning UI:** Frosted-glass aesthetics with customizable wallpapers (Images & GIFs).
- **Productivity Tools:** Built-in Focus Timer, To-Do List, and Quick Notes etc.
- **Smart Search:** Central search bar compatible with Google, Perplexity, ChatGPT, and more.
- **Zen Mode:** A minimalist, full-screen clock to help you disconnect.
- **Privacy First:** All data (notes, tasks, links) is stored locally on your device.
- and many more...

## üîí Privacy Policy

**Your data is yours.**
Glassy stores all your settings, tasks, and notes locally using Chrome's storage API. We do not track, collect, or sell your personal data.
